# Structure

- A document is a collection of arguments (probably) around a theme.
- An argument contains the following:
    - Claim 
        - Theorem
        - Lemma
        - Remark?
    - Ground
        - Definition
        - Axiom
        - Evidence
    - Warrant
        - Proof
        - Corollary?
        - Algorithm?
- The ground and warrant are conditional.
